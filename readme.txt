La aplición ha sido realizada con el Framework Phaser en su totalidad.
Las tres secciones, Instrucciones, juego y resultados han sido creadas en el Canvas mediante Phaser para una mejor adaptación a dispositivos móviles.
El juego realizado es responsive, adaptandose a cualquier tipo de pantalla en posición landscape.
La app ha sido generada sólo para android (no dispongo de un mac para desarrollar correctamente en ios) utilizando Android Studio con Xwalk. En éste caso las 
imágenes se destruyen al tocarlas.
He tenido problemas a la hora de integrar la API 500px en la aplicación por lo que al final no la he incluido aunque las imagenes utilizadas han sido sacadas de ésta página.
